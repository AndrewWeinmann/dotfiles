# Starship Configuration

Starship prompt with a minimal baseline and an OS/WSL indicator.

## What's Included

- **`starship.toml`**: Main configuration, including a Windows/WSL indicator

## Setup

Run the parent `install.sh` or directly run `./setup.sh` in this directory.

The setup process will:

1. Create or symlink `starship.toml` to `~/.config/starship.toml`
2. Back up any existing config file
